## 简单易懂的观察者模式

### 一、什么是观察者模式

观察者模式建立了一套触发机制，当对象之间存在一对多的依赖关系时，其中一个对象的状态发生改变，所有依赖它的对象都会收到通知，这就是观察者模式。

### 二、实际场景

#### 1. DOM事件

```javascript
document.body.addEventListener('click', () => {
    console.log(1);
}, false)
```
